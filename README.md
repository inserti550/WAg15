
# WAg15
**WAg15** - Winamp plugin for Logitech monochrome LCD, that allows you to control the player using soft buttons and view player statistics on the LCD screen

> Supports Russian and English output languages.

> Now support WACUP x86 and x64.

**Latesd release:** https://github.com/Nazar-166/WAg15/releases/

## At the moment the plugin has
1. Display song title
2. Display elapsed and total song time
3. Progress bar
4. Settings menu
5. Control for winamp
6. Realtime preview

## Possible action
  - None
  - Next Track
  - Previous Track
  - Play/Pause
  - Stop
  - Fast-forward 5s
  - Rewind 5s
  - Jump to Start
  - Jump to End
  - Toggle Repeat
  - Toggle Shuffle
  - Volume Up
  - Volume Down
  - Set Volume 0%
  - Set Volume 25%
  - Set Volume 50%
  - Set Volume 75%
  - Set Volume 100%

## Installing
1. Download the compiled "gen_g15.dll" from the [Releases](https://github.com/Nazar-166/WAg15/releases)
2. Log in to Winamp, go to the "Plug-ins" section, there you will see the path by which your Winamp recognizes plugins (default this "C:\Program Files (x86)\Winamp\Plugins\")
3. Put file in your Winamp plugins folder
4. Restart Winamp, if it's running
5. Congratulation! you install this plugin. Configure the plugin for yourself in the General Purpose section -> G15 plugin!
